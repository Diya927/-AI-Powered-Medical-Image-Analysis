## 📋 Project Overview
This project implements a deep learning pipeline to analyze chest X-ray images and identify signs of Pneumonia. By leveraging **Convolutional Neural Networks (CNNs)** and automated image preprocessing, the system acts as a "Virtual Doctor" to provide rapid diagnostic support.

## 🚀 Key Features
- **End-to-End Pipeline:** Automated dataset ingestion from Kaggle and real-time preprocessing.
- **High Performance:** Achieved a validation accuracy of **91.2%**.
- **Secure Deployment:** Implemented industry-standard credential management for API access.
- **Visual Analytics:** Real-time generation of Confusion Matrices and diagnostic reports.

## 🛠️ Tech Stack
- **Languages:** Python
- **Deep Learning:** TensorFlow, Keras
- **Computer Vision:** OpenCV, Pillow
- **Data Analysis:** NumPy, Pandas, Scikit-learn
- **Visualization:** Matplotlib, Seaborn

## 📊 Results & Evaluation
The model was trained on the *Chest X-Ray Images (Pneumonia)* dataset. Below is the performance summary:

- **Validation Accuracy:** 91.2%
- **Loss:** Binary Cross-entropy
- **Output:** Virtual Doctor Report (Diagnostic Classification)
