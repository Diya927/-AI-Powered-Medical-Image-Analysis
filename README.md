# 🏥 AI-Powered Medical Image Analysis System

## 📌 Project Overview
This is an industry-oriented AI diagnostic tool built to detect Pneumonia from Chest X-Rays. It simulates a real-world radiology assistance system, reducing diagnosis time from 24 hours to under 10 seconds.

## 🛠️ Tech Stack & Workflow
- **AI Framework:** TensorFlow / Keras
- **Image Processing:** OpenCV
- **Workflow:** 
  1. Data ingestion of 2.3 GB Kaggle dataset.
  2. Preprocessing & Grayscale conversion.
  3. CNN training (Convolutional Neural Network).
  4. Accuracy evaluation and Confusion Matrix analysis.
  5. Virtual Simulation for real-time diagnosis reporting.
